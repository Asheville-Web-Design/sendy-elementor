Easily connect Elementor Pro forms to Sendy and automatically grow your email list with just a few clicks—no third-party tools required.

== Description ==
Are you tired of manually adding subscribers to your Sendy email lists? The Integration Sendy for Elementor plugin eliminates the hassle by seamlessly connecting your Elementor Pro forms to Sendy. With a quick and easy setup, your email marketing efforts can run on autopilot.

**Key Features:**
- **Effortless Setup:** Integrate in minutes—just add your Sendy API Key and List ID.
- **Form Field Mapping:** Map Elementor form fields to match name and email fields in Sendy.
- **Automation Ready:** Automatically add new form submissions to your Sendy email list.
- **No Third-Party Services:** The plugin directly connects Elementor and Sendy, keeping your data secure and your setup simple.

**How It Works:**
1. **Install & Activate the Plugin:** Upload the plugin to your WordPress site and activate it.
2. **Configure the Settings:** Go to Settings > Sendy Elementor. Enter your Sendy API Key, List ID, and map form fields.
3. **Automate Your List Building:** Your Elementor forms will now automatically send new subscribers to your Sendy lists!

**Who Is This For?**
- Website owners who use Elementor Pro and Sendy for email marketing.
- Businesses looking to automate their lead generation process.
- Marketers wanting a simple, direct integration without needing additional tools.

== External Services ==
This plugin integrates Elementor Pro forms with Sendy, a self-hosted email newsletter application.

**Data Sent:**
- User's name and email address (from the Elementor form fields)
- Sendy List ID (entered in the plugin settings)
- API key (entered in the plugin settings)

**Where Is the Data Sent?**
- Data is sent to the Sendy installation URL you configure in the plugin settings.

== Installation ==
1. Upload the `integration-sendy-elementor` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Settings > Sendy Elementor** and enter your API Key, List ID, and map form fields.
4. Use any Elementor Pro form on your site to start building your email list automatically.

== Frequently Asked Questions ==
= Do I need Elementor Pro for this plugin? =
Yes, the plugin works specifically with Elementor Pro because it utilizes the Form Widget.

= What do I need to connect to Sendy? =
You'll need your Sendy API Key and the List ID of the email list where you want new subscribers added.

= Is my data safe with this plugin? =
Absolutely. No data is sent to third-party servers. Data is securely transferred directly from your site to your Sendy installation.
